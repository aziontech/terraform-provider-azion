# Security Policy

## Supported Versions

Only the latest version on the default branch is supported with security updates.

## Reporting a Vulnerability

Report vulnerabilities to **security@azion.com** with:

1. Description of the vulnerability
2. Steps to reproduce
3. Potential impact
4. Suggested fix (if any)

You will receive an acknowledgement within **2 business days** and a detailed response within **5 business days** indicating next steps.
