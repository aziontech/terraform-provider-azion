# azion_digital_certificate (Data Source)

Provides a data source to read a specific digital certificate by its identifier.

## Example Usage

```terraform
# Read a specific digital certificate by ID
data "azion_digital_certificate" "example" {
  certificate_id = 12345
}

# Output the certificate details
output "certificate_name" {
  value = data.azion_digital_certificate.example.results.name
}

output "certificate_status" {
  value = data.azion_digital_certificate.example.results.status
}
```
