# azion_application_device_groups (Data Source)

Provides an application device groups data source for listing all device groups in an application. Device groups allow you to categorize user agents (browsers, devices) using regular expression patterns.

## Example Usage

### List All Device Groups for an Application

```terraform
data "azion_application_device_groups" "example" {
  application_id = 12345
}
```

### List Device Groups and Output Results

```terraform
# List all device groups for an application
data "azion_application_device_groups" "all" {
  application_id = 12345
}

# Output the total count of device groups
output "device_groups_count" {
  value = data.azion_application_device_groups.all.counter
}

# Output the first device group name
output "first_device_group_name" {
  value = data.azion_application_device_groups.all.results[0].name
}
```
