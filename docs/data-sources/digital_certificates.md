# azion_digital_certificates (Data Source)

Provides a data source to list all digital certificates in your Azion account.

## Example Usage

```terraform
# List all digital certificates
data "azion_digital_certificates" "all" {}

# Output the count of certificates
output "certificates_count" {
  value = length(data.azion_digital_certificates.all.results)
}

# Output all certificate names
output "certificate_names" {
  value = [cert for cert in data.azion_digital_certificates.all.results : cert.name]
}
```
