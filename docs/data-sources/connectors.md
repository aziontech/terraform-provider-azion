# azion_connectors (Data Source)

Provides a data source for listing all Connectors in the account. Connectors are polymorphic and support different types (http, storage).

## Example Usage

### List All Connectors

```terraform
data "azion_connectors" "all" {}

# Output the count
output "total_connectors" {
  value = data.azion_connectors.all.counter
}

# Output all connector names
output "connector_names" {
  value = [for c in data.azion_connectors.all.results : c.name]
}

# Output all connector types
output "connector_types" {
  value = [for c in data.azion_connectors.all.results : c.type]
}
```

### Filter Connectors by Type

```terraform
data "azion_connectors" "all" {}

# Get all HTTP connectors
locals {
  http_connectors = [for c in data.azion_connectors.all.results : c if c.type == "http"]
}

output "http_connector_names" {
  value = [for c in local.http_connectors : c.name]
}
```
