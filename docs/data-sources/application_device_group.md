# azion_application_device_group (Data Source)

Provides an application device group data source for reading a single device group by ID. Device groups allow you to categorize user agents (browsers, devices) using regular expression patterns.

## Example Usage

### Read a Device Group by ID

```terraform
data "azion_application_device_group" "example" {
  application_id = 12345
  id             = "67890"
}
```

### Read a Device Group from a Resource Reference

```terraform
# Read the device group using its ID
data "azion_application_device_group" "by_id" {
  application_id = 12345
  id             = "67890"
}

# Output the device group name
output "device_group_name" {
  value = data.azion_application_device_group.by_id.data.name
}
```
