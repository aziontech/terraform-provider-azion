# azion_application_device_group (Resource)

Creates an application device group resource. Device groups allow you to categorize user agents (browsers, devices) using regular expression patterns.

## Example Usage

### With Parent Application

```terraform
# First, create the parent application
resource "azion_application_main_setting" "example" {
  application = {
    name   = "My Application"
    active = true
  }
}

# Then create the device group for that application
resource "azion_application_device_group" "mobile" {
  application_id = azion_application_main_setting.example.application.application_id
  device_group = {
    name       = "mobiledevices"
    user_agent = ".*(Mobile|Android|iPhone).*"
  }
}
```

### Create a Device Group for Mobile Devices

```terraform
resource "azion_application_device_group" "mobile" {
  application_id = 12345
  device_group = {
    name       = "mobiledevices"
    user_agent = ".*(Mobile|Android|iPhone).*"
  }
}
```

### Create a Device Group for Tablets

```terraform
resource "azion_application_device_group" "tablet" {
  application_id = 12345
  device_group = {
    name       = "tablets"
    user_agent = ".*(iPad|Tablet|Kindle).*"
  }
}
```

### Create Multiple Device Groups

```terraform
# Mobile devices group
resource "azion_application_device_group" "mobile" {
  application_id = 12345
  device_group = {
    name       = "mobile"
    user_agent = ".*(Mobile|Android|iPhone).*"
  }
}

# Desktop browsers group
resource "azion_application_device_group" "desktop" {
  application_id = 12345
  device_group = {
    name       = "desktop"
    user_agent = ".*(Windows NT|Macintosh|Linux).*"
  }
}
```

## Import
